User Story

我是一個網站管理者，我希望我的文章容易被看見，提升網站流量.

公司專案推廣AMP

  • 需求 : 將原有文章HTML格式轉換成支援AMP的格式.
  • 實作 : 後台寫支Helper，儲存同時儲存一份AMP格式.
  • 目標 : Helper 抽成package.
